## Phone Addiction Predictor 

## Problem :
Excessive phone usage is becoming a major issue among students, affecting sleep, productivity, and mental health.

## Solution :
This project uses a Machine Learning model to predict whether a person is addicted to their phone based on:
1. Screen Time
2. Sleep Hours
3. App Usage

## Technologies Used :
1. Python
2. Pandas
3. Scikit-learn

## How to Run :
1. Install dependencies:
   pip install pandas scikit-learn

2. Run the program:
   python main.py

## Output :
The model prints:
- Accuracy of prediction
- Whether a person is addicted (1) or not (0)

## Files :
1. main.py --> ML model
2. data.csv --> dataset
3. README.md --> documentation
